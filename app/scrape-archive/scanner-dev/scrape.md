# Scrape Analysis: Scanner.dev

- **Font stack:** Geist Sans, Geist Mono. This is the defining typographic choice.
- **Color tokens:** 
    - Background: #000000
    - Text: #FFFFFF (Primary), #A1A1AA (Secondary)
    - Accent: #00F0FF (Cyan/Aqua) for primary buttons
    - Border: #27272A
- **Spacing system:** Modular, grid-based. Consistent 24px and 48px gaps.
- **Card/tile layouts:** High-density feature cards with "arrow_right" indicators.
- **Search/query UX:** "Search & Investigate" is a core pillar. Mentions "Lightning-fast full-text search".
- **Navigation patterns:** Clean top-level links (Product, Resources). "Book a Demo" CTA.
- **Hero treatment:** Bold headings: "Your SIEM sees 1%. Scanner sees what it can't." High-contrast, minimal noise.
- **Category tag:** [PIPELINE]

### Comparison to Panther Reference
- Panther uses a similar dark/terminal aesthetic but Scanner.dev feels more "modern SaaS" via Geist fonts.
- kjtcom will adopt Geist Sans/Mono as the primary font choice based on this site.
